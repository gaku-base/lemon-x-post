# ポストコンシェルジュ v4.10.7

和洋喫茶 レモンの木向けの、X投稿文作成アプリです。

## 主な機能

- Airリザーブの予約状況取得
- 午前・午後を分けた空席案内
- 津市の天気情報取得
- メニューを反映した投稿文生成
- X公式カウント220〜250を目標に自動調整
- 最大270カウント、超過時は自動短文化
- iPhoneホーム画面アイコン対応

## フォルダー構成

- `.github/` GitHub Actions
- `assets/` 現行ロゴとアイコン
- `data/` Airリザーブ取得結果
- `scripts/` Airリザーブ取得処理
- `vendor/` twitter-textとライセンス
- `app-*.js` アプリ機能
- `styles-*.css` 画面デザイン
- `index.html` アプリ画面
