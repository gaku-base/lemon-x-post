# ポストコンシェルジュ v4.11.0

和洋喫茶 レモンの木向けの、X投稿文作成アプリです。

## 主な機能

- Airリザーブの予約状況取得
- 午前・午後を分けた空席案内
- 津市の天気情報取得
- メニューを反映した投稿文生成
- X公式カウント220〜250を目標に自動調整
- 最大270カウント、超過時は自動短文化
- iPhoneホーム画面アイコン対応

## フォルダー構成

- `.github/` GitHub Actions
- `assets/` 現行ロゴとアイコン
- `data/` Airリザーブ取得結果
- `scripts/` Airリザーブ取得処理
- `vendor/` twitter-textとライセンス
- `app-*.js` アプリ機能
- `styles-*.css` 画面デザイン
- `index.html` アプリ画面

## v4.10.8

当日のAirリザーブ取得時刻が、当日の最終予約枠終了時刻の3時間前以降で、
取得結果が「予約枠公開前」になった場合は「営業終了」としてグレー表示します。
翌日以降、満席、空きあり、定休日の判定は変更しません。


## v4.10.9

投稿対象日と投稿日との差に応じて、投稿文の書き出しを自動変更します。

- 当日：本日も営業いたします。
- 明日：明日も営業いたします。
- 明後日：明後日も営業いたします。
- 3日後：3日後も営業いたします。
- 4日後以降：対象日の日付を入れて案内します。

X向け自動短文化でも、この日付に対応した書き出しは削除しません。


## v4.10.10

投稿文の書き出しを次のルールへ変更しました。

- 当日：本日も営業いたします。
- 明日：明日も営業いたします。
- 明後日以降：対象日の日付から書き始めます。

例：
- 明後日が7/19（日）の場合：7/19（日）も営業いたします。
- 3日後が7/20（月）の場合：7/20（月）も営業いたします。

X向け自動短文化でも、この日付に対応した書き出しは保持します。


## v4.11.0

投稿文作成エリアを「X用」と「Instagram用」のタブに分けました。

### X用
- 220〜250カウントを目標
- 最大270カウント
- X向け自動短文化
- X投稿画面を開く

### Instagram用
- メニュー、天気、予約状況を段落分け
- 2,200文字までの文字数表示
- レモンの木向け固定定型文
- メニューに応じたハッシュタグ
- Instagramを開く

選択したタブは端末に保存され、次回起動時にも維持されます。
